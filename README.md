# Projeto Agrinho

## Objetivo

Desenvolver uma página utilizando HTML, CSS e JavaScript para apresentar a importância do programa Agrinho e destacar a conexão entre o campo e a cidade, mostrando como ambos dependem um do outro para o desenvolvimento da sociedade.

## Justificativa

Este projeto foi criado como atividade da disciplina para aplicar os conhecimentos em HTML, CSS e JavaScript. Além disso, busca conscientizar sobre a importância do campo na produção de alimentos, da cidade no desenvolvimento tecnológico e da sustentabilidade como responsabilidade de todos.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Página informativa sobre o Agrinho.
- Estilização com cores relacionadas à natureza.
- Botão interativo que exibe uma curiosidade sobre a agricultura.
